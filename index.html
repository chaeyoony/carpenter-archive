지도 데이터를 불러오는 중입니다...

지도 데이터를 불러오지 못했어요.

이 파일을 컴퓨터에서 직접 열면 브라우저 보안 정책(CORS) 때문에 외부 지도 데이터를 불러올 수 없어요. 다음 중 한 가지 방법으로 해결할 수 있습니다.

  1. Netlify Drop에 배포 (가장 간단) · app.netlify.com/drop 에 이 HTML 파일을 끌어다 놓으면 즉시 인터넷 주소가 생깁니다.
  2. 로컬 서버 실행 · 터미널에서 파일이 있는 폴더로 이동 후 python3 -m http.server 8000 → 브라우저에서 http://localhost:8000 열기.
  3. VS Code의 Live Server 확장으로 열기.

※ 인터넷 연결이 필요한 이유: 실제 대한민국 행정구역 경계 데이터를 오픈 데이터에서 받아오기 때문입니다.
